# 시작하기 (클로드 코드에서 대화로)

이 도구는 **클로드 코드(Claude Code)** 스킬입니다. 어려운 명령어를 직접 칠 필요가 없어요.
**클로드 코드에게 말로 시키면** 클로드가 알아서 설치·배포·검증을 해줍니다. 여러분은 **구글 웹에서
클릭해야 하는 몇 가지**만 하면 됩니다.

## 전체 순서 (한눈에)

1. **준비물** — 구글 계정 1개 + 클로드 코드
2. **스킬 설치** — 폴더 넣기 (한 번만)
3. **대화 시작** — 클로드 코드에 "디맨드젠 셋업해줘" 라고 입력
4. **웹 작업만 대신** — 클로드가 링크를 주면 브라우저에서 하고 "했어" 라고 답
5. **완료** — "준비 완료 ✅" 뜨면 끝. 이후엔 대화로 광고 설정

아래에서 하나씩 따라 하세요.

---

## 1단계 · 준비물

- **구글 계정(Gmail) 1개** — 처음부터 끝까지 **이 계정 하나로만** 진행합니다. 없으면
  [accounts.google.com/signup](https://accounts.google.com/signup)에서 만드세요.
- **클로드 코드** — 지금 이 글을 클로드 코드로 보고 있다면 이미 준비된 거예요.

## 2단계 · 스킬 설치 (한 번만)

클로드 코드가 이 도구를 쓸 수 있게 폴더를 넣어 줍니다.

### 가장 쉬운 방법 — 클로드에게 부탁하기

클로드 코드 입력창에 이렇게 써 보세요:

```text
이 깃허브 스킬을 내 클로드 코드에 설치해줘:
https://github.com/aiebrain/gads-demandgen-setup
```

### 직접 넣는 방법

1. [github.com/aiebrain/gads-demandgen-setup](https://github.com/aiebrain/gads-demandgen-setup)
   접속 → 초록색 **`< > Code`** → **Download ZIP** → 압축 풀기
2. 압축 푼 폴더 안 `skills` 폴더에 있는 **폴더 2개**를 복사합니다:
   `google-ads-demandgen-setup` 와 `google-ads-direct-mcp`
3. 내 PC의 **`.claude/skills`** 폴더에 붙여넣기
   - Windows: `C:\Users\<내이름>\.claude\skills\`
   - Mac: `~/.claude/skills/`
   - (그 폴더가 없으면 새로 만드세요.)
4. 클로드 코드를 새로 시작하거나 새 대화를 엽니다.

## 3단계 · 대화로 셋업 시작

클로드 코드에 이렇게 말하면 됩니다:

```text
디맨드젠 광고 설정 환경을 셋업해줘
```

그러면 클로드가 스킬을 실행해서 **환경 진단 → (필요한 것 물어보기) → 자동 설치·배포·등록·검증**까지
알아서 진행합니다. 중간에 필요한 정보를 물어보면 답해 주세요.

## 4단계 · 클로드가 부탁하는 "웹 작업"만 하기

구글 정책상 자동으로 안 되는 것들(로봇 방지·심사 등)은 **클로드가 링크와 방법을 알려 줍니다.**
여러분은 브라우저에서 그것만 하고 **"했어"** 라고 답하면 돼요. 대표적으로:

- 관리자(MCC) 계정 만들기
- 개발자 토큰 신청
- OAuth 동의 화면에서 **"허용"** (그리고 "테스트 사용자"에 내 이메일 추가)

각각의 **클릭 하나하나 방법**은 [셋업 가이드 (0~4단계)](/setup)에 자세히 있습니다.

## 5단계 · 완료

클로드가 **"디맨드젠 광고 설정 준비 완료 ✅"** 라고 하면 끝입니다. 이후에는 대화로 시키면 돼요:

```text
PAUSED 상태로 디맨드젠 캠페인 하나 만들어줘
```

> 💰 캠페인을 실제로 켜면 **광고비가 나갑니다.** 그래서 새 캠페인은 항상 **PAUSED(일시중지)** 로 만들고,
> 켜기 전에 반드시 다시 확인합니다.

---

## (고급) 클로드 코드 없이 터미널로 직접 하기

클로드 코드를 안 쓰고 직접 실행하려면, 터미널을 `skills/google-ads-demandgen-setup/scripts` 폴더에서
열고 아래를 실행합니다:

```bash
python guided_setup.py   # 계정·프로젝트 준비 안내
python bootstrap.py      # 설치·배포·등록·검증
```

터미널 여는 법이 낯설면 대부분은 그냥 **클로드 코드에게 시키는 편**이 훨씬 쉽습니다.
